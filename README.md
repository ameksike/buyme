# Buyme
Demo E-Comerce App using Nextjs v15, React v19, etc.

## Install 
- terraform
    - terraform -version
    - terraform init
    - terraform plan
    - terraform apply
    - terraform destroy
- [Frontend](./frontend/README.md)

## Run 
- [Vercel Dashboard](https://vercel.com/antonio-membrides-espinosas-projects/buyme)
- [Web Demo](https://buyme-tau.vercel.app/)
